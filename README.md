# Apresentação

Plataforma web estática para GitHub Pages com foco no estudo acadêmico do SBSJ.

## Páginas

- `index.html`: visão geral e navegação da ferramenta.
- `mapa.html`: mapa interativo com divisões temáticas.
- `formulario.html`: formulário técnico por tópicos do estudo.
- `relatorio.html`: estrutura base para redação do relatório acadêmico.
