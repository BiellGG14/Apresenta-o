# Geodados

Pasta histórica do projeto para dados espaciais publicados no mapa.

Apesar do nome, o mapa atual usa KML/KMZ úteis em `sbsj/originais/`.
Superfícies e EAC em GPKG/SHP são mantidos como fonte e publicados no mapa por KML derivado.
Novos GeoJSON só devem ser criados se houver necessidade técnica clara.
