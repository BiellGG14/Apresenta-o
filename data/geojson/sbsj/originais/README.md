# Originais úteis do SBSJ

Esta pasta guarda apenas os KML/KMZ úteis promovidos da fonte bruta local para uso rastreável no mapa.

O mapa carrega estes arquivos diretamente no navegador:

- `opeaSBSJ.kml`
- `pbzph.kml`
- `pbzpa_SBSJ.kmz`
- `pzpana_SBSJ.kmz`

Arquivos da pasta bruta `data/02-Dados Geo/` não devem ser carregados diretamente pelo site.
